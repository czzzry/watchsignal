# Triage Labels

The engineering skills use five canonical triage roles.
This file maps those roles to the label strings this repo expects to use in GitHub Issues.

| Canonical role | Label in this repo | Meaning in this repo |
|---|---|---|
| `needs-triage` | `needs-triage` | Founder or maintainer needs to evaluate the item |
| `needs-info` | `needs-info` | Waiting on founder or reporter for more information |
| `ready-for-agent` | `ready-for-agent` | Fully specified and ready for an agent to pick up |
| `ready-for-human` | `ready-for-human` | Founder review, judgment, or approval is needed before the next step |
| `wontfix` | `wontfix` | Will not be actioned |

When a skill refers to one of these roles, use the mapped label string from this table.
